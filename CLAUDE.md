# CLAUDE.md

This file contains guidelines for Claude Code when working on this project.

## Project description

Personal static landing page for GitHub Pages. Clean resume clone.

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS

## Conventions

- Use **pnpm** for everything. Do not use npm or yarn under any circumstances
- TypeScript is mandatory
- Always use Tailwind CSS for styling
- Prefer ESM and modern browser syntax

## UI and styling

- Tailwind is the only styling solution
- Prioritize readability over visual micro-optimizations
